# Last Heart - Minecraft Mod 💔

Java Minecraft uchun hardcore rejimi modifikatsiyasi. Oyunchi o'lib qolsa, bitta yurak yo'qoladi. Maksimal 20 ta yurakga (10 odatiy + 10 qo'shimcha) yetishi mumkin.

## 🎮 Xususiyatlar

### Yurak Tizimi
- **Default Hearts**: 10 ta yurak (2 tan satir)
- **Extra Hearts**: 0-10 ta qo'shimcha yurak
- **Maximum**: 20 ta yurak (10 satir)
- **Son'iy o'lim**: O'yin tugaydi

### Crafting Recipe
Heart-Soul yasash recipe:
```
Netherite Ingot  | Wither Skull      | Netherite Ingot
Echo Shard       | Nether Star       | Echo Shard
Netherite Ingot  | Heart-Soul        | Netherite Ingot
```

### Half-Heart Drops (Mobs)
- **Wither**: 20%
- **Ender Dragon**: 100%
- **Elder Guardian**: 25%
- **Ravager**: 10%
- **Warden**: 25%

### Heart-Tier Drops (Mobs)
- **Iron Golem**: 0.5%
- **Witch**: 5%
- **Wither Skeleton**: 15%
- **Piglin Brute**: 10%
- **Vindicator**: 10%
- **Evoker**: 10%
- **Enderman**: 1%

## 🎯 Kontrol

### Yurak Qo'shish
- **Right Click**: Yurakni o'ziga qo'shish

### Yurak Olib tashlash
- **Ctrl + Shift + Right Click**: Do'stlardan yurakni olib olish (multiplayer)

## 🎵 Effektlar

- 🔴 **Red Screen**: 1 ta yurak qolganida oynaning ekrani qizarib qoladi
- 🎵 **Sound Effects**: Yurak o'zgarganida ovoz chiqadi
- ✨ **Particle Effects**: Visual effektlar

## 📦 Totem of Undying
- Standart Minecraft mexanizmasi kabi ishlaydi
- Totem ishlatilsa default yurak qayta tiklanadi
